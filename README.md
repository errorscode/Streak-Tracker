# Streak Tracker for Obsidian

**Streak Tracker** is an Obsidian plugin that helps you maintain a daily usage streak. It tracks how many consecutive days you use Obsidian and resets the streak if you miss a day.

---

## Features

- Tracks daily usage streak.
- Automatically resets streak if a day is missed.
- Shows notifications when your streak continues or is reset.
- Settings tab allows you to see your current streak and manually reset it.
